# Simulation de modèles linéaires

## Objectif
Ce projet analyse des données simulées pour étudier le comportement des estimateurs de régression linéaire sous différentes hypothèses.
Il analyse l'impact de variables sur les résultats d'étudiants et predit leurs scores aux examens.

## Technologies utilisées
- R
- ggplot2 pour les visualisations

## Contenu
- Simulation de données
- Comparaison de modèles
- Tests d'hypothèses
- Prédiction de notes d'étudiants
- Compte rendu des résultats

## Comment exécuter
1. Ouvrir `runanalysis.R` dans RStudio
2. Installer les packages : `install.packages(c("ggplot2", "dplyr"))`
3. Exécuter le script ligne par ligne
